[edit]
 
bool unique() const noexcept;
 (deprecated in C++17)

Checks if *this is the only shared_ptr instance managing the current object, i.e. whether use_count() == 1.

Contents

 [hide] 
  • 1 Parameters
  • 2 Return value
  • 3 Notes
  • 4 Example
  • 5 See also

[edit] Parameters

(none)

[edit] Return value

true if *this is the only shared_ptr instance managing the current object, false otherwise.

[edit] Notes

This function is deprecated as of C++17 because use_count is only an approximation in multithreaded environment (see Notes in use_count)

[edit] Example

Run this code
#include <memory> 
#include <iostream> 
 
int main() 
{ 
    auto sp1 = std::make_shared<int>(5);
    std::cout << std::boolalpha;
    std::cout << "sp1.unique() == " << sp1.unique() << '\n'; 
 
    std::shared_ptr<int> sp2 = sp1; 
    std::cout << "sp1.unique() == " << sp1.unique() << '\n'; 
}

Output: 

sp1.unique() == true
sp1.unique() == false
