[edit]
 

This header was originally in the C standard library as <inttypes.h>.

Provides conversions from C byte and wide strings to std::intmax_t and std::uintmax_t, overloads some math functions for std::intmax_t and provide C style input/output format macros for the types declared in <cstdint>.
